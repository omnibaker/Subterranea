
# SUBTERRANEA
*A 2D game demo by Stephen Baker*
 
### ABOUT
This game is based loosly on a tutorial found on Udemy Course *Complete C# Unity Game Developer 3D* by GameDev.tv. Conceptually, the gameplay remains the same but the architecture, controls, and artwork are of my own creation. Third-party code cited where necessary. Sounds are sourced from *Interface and Item Sounds* by Cafofo, purchased at Unity's Asset Store.

### PLAY
- Load from Main scene, this opens up the StartMenu scene automatically. 
- Pressing the start button here will open the Play scene.
- If on a desktop, hold the **SPACE** bar to thrust forward. Release to stop. Press **A**/**D** keys to pivot left/right and change direction.
- On iOS/Android devices, On-Screen button will appear. Press the round button on the right is to thrust. Press the two arrow buttons on the left to pivot left/right and change direction.
- The aim of each cave is for player to fly from starting platform (indicated with flashing red light) to end platform (indicated with green flashing lights).

### SETUP
- Clone/download repository and add to Unity Hub menu.
- Compatible with for Unity 2021.3.4f1.
- Tested on iOS, Android and Windows 11.
